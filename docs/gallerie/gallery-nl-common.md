### *Brassicaceae*-type

{{ gallery("brassica_napus") }}

- Pollenklasse: tricolpaat
- Vorm: rond
- Grootte: 26 µm (22.3-28.2 µm)
- Structuur: reticulaat
- Voorbeeld: *Brassica napus* (koolzaad)
- Familie: Brassicaceae (kruisbloemenfamilie)

### *Trifolium repens*-type

{{ gallery("trifolium_repens") }}

- Pollenklasse: tricolporaat
- Vorm: driehoekig convex
- Grootte: 21-25 µm
- Structuur: fijn reticulaat volgens Beug; LM: psilaat, EM: punctaat volgens PalDat
- Voorbeeld: *Trifolium repens* (witte klaver)
- Familie: Fabaceae (vlinderbloemenfamilie)

### *Salix*-type

{{ gallery("salix_caprea") }}

- Pollenklasse: tricolpaat (soms met duidelijke porie)
- Vorm: rond tot licht driehoekig
- Grootte: 17-26 µm
- Structuur: reticulaat (mazes kleiner naar de colpi)
- Voorbeeld: *Salix caprea* (boswilg / waterwilg)
- Familie: Salicaceae (wilgenfamilie)

### Taraxacum-type

{{ gallery("taraxacum_officinale") }}

- Pollenklasse: tricolpaat (3-4 slecht zichtbare aperturen)
- Vorm: rond tot driehoekig, sferoid
- Grootte: 28 µm
- Structuur: fenestraat, echinaat
- Voorbeeld: *Taraxacum officinale* (paardenbloem)
- Familie: Asteraceae (composietenfamilie)

### Rubus-type

{{ gallery("rubus_fruticosus") }}

- Pollenklasse: tricolpaat
- Vorm: driehoekig tot vierhoekig
- Grootte: 28.6-36.9 µm
- Structuur: rugulaat
- Voorbeeld: *Rubus fruticosus* (gewone braam)
- Familie: Rosaceae (rozenfamilie)

### *Tilia*-type

{{ gallery("tilia_platyphyllos") }}

- Pollenklasse: tricolporaat
- Vorm: rond tot driehoekig; oblaat
- Grootte: 32-38 µm
- Structuur: reticulaat tot fossulaat
- Voorbeeld: *Tilia platyphyllos* (grootbladige linde)
- Familie: Malvaceae (kaasjeskruidfamilie)

### Prunus/Pyrus-type

{{ gallery("prunus_avium") }}

- Pollenklasse: tricolporoidaat / tricolporaat (Beug 20, 20.12 Sorbus-groep)
- Vorm: driehoekig; licht oblaat tot sferoid
- Structuur: striaat (Prunus duidelijk striaat; Pyrus striaat/rugulaat tot verrucaat/scabraat)
- Voorbeelden:
  - *Prunus domestica* (pruim): 44 µm (38.9-48.6 µm)
  - *Pyrus communis* (peer): 32 µm (29.2-36.0 µm)
- Familie: Rosaceae (rozenfamilie)

### *Castanea*

{{ gallery("castanea_sativa") }}

- Pollenklasse: tricolporaat
- Vorm: ovaal
- Grootte: 12 µm–14 µm
- Structuur: glad
- Voorbeeld: *Castanea sativa* (tamme kastanje)
- Familie: Fagaceae (napjesdragersfamilie)

### *Phacelia*

{{ gallery("phacelia_tanacetifolia") }}

- Pollenklasse: stephanocolpaat tot heterocolpaat (Beug 24); 6 colpi, afwisselend breed en smal
- Vorm: rond tot rozetvormig, sferoid tot optisch licht oblaat
- Grootte: 20 µm (18.8-23.1 µm)
- Structuur: scabraat tot psilaat; apertuurmembranen licht korrelig
- Voorbeeld: *Phacelia tanacetifolia* (phacelia, bijenvoer)
- Familie: Boraginaceae (ruwbladigenfamilie)

### *Robinia*

{{ gallery("robinia_pseudoacacia") }}

- Pollenklasse: tricolpaat (soms tetracolporoidaat)
- Vorm: driehoekig tot rond, sferoid tot optisch gering oblaat
- Grootte: 30 µm
- Structuur: scabraat; aperturen zonder ornamentatie
- Voorbeeld: *Robinia pseudoacacia* (valse acacia)
- Familie: Fabaceae (vlinderbloemenfamilie)

### *Trifolium pratense*

{{ gallery("trifolium_pratense") }}

- Pollenklasse: tricolporaat
- Vorm: gering driehoekig convex
- Grootte: 35.9-41.5 µm
- Structuur: grof reticulair
- Voorbeeld: *Trifolium pratense* (rode klaver)
- Familie: Fabaceae (vlinderbloemenfamilie)

### *Ligustrum*

{{ gallery("ligustrum_vulgare") }}

- Pollenklasse: tricolpaat tot tricolporaat
- Vorm: rond tot licht oblaat
- Grootte: ca. 24-34 µm (indicatief ca. 28×34 µm)
- Structuur: reticulaat
- Voorbeeld: *Ligustrum vulgare* (wilde liguster)
- Familie: Oleaceae (olijffamilie)

### *Calluna*

{{ gallery("calluna_vulgaris") }}

- Pollenklasse: tetrade (Beug 4); korte brede colpi, tricolpaat of tricolporaat
- Vorm: tetrade, sferoid
- Grootte: 32-39 µm
- Structuur: psilaat
- Voorbeeld: *Calluna vulgaris* (struikheide)
- Familie: Ericaceae (heidefamilie)

### *Fagopyrum*

{{ gallery("fagopyrum_esculentum") }}

- Pollenklasse: tricolporaat (poren vaak niet te zien)
- Vorm: rond, sferoid
- Grootte: 51 µm
- Structuur: reticulaat; aperturen met korrelige ornamentering
- Voorbeeld: *Fagopyrum esculentum* (boekweit)
- Familie: Polygonaceae (duizendknoopfamilie)

### *Helianthus*

{{ gallery("helianthus_annuus") }}

- Pollenklasse: tricolporaat
- Vorm: rond
- Grootte: 35 µm
- Structuur: echinaat H
- Voorbeeld: *Helianthus annuus* (zonnebloem)
- Familie: Asteraceae (composietenfamilie)

### *Rhododendron*

{{ gallery("rhododendron_ponticum") }}

- Pollenklasse: tetrade
- Vorm: tetrade
- Grootte: 36-50 µm
- Structuur: verrucaat
- Voorbeeld: *Rhododendron ponticum* (rhododendron)
- Familie: Ericaceae (heidefamilie)
