## Brekingsindex


| **Material / Resin**    | **Refractive Index (nD​)** | **Best Use for Pollen**               |
| ----------------------- | -------------------------- | ------------------------------------- |
| **Water**               | 1.33                       | Living/hydrated pollen (Low contrast) |
| **Glycerine Jelly**     | 1.40–1.47                  | Standard mounting (Good contrast)     |
| **UV Resin (Standard)** | **1.49–1.52**              | **High contrast for surface exine**   |
| **Pollen Exine**        | **1.53–1.58**              | The target material                   |
| **UV Resin (Optical)**  | **1.55–1.57**              | **"Clearing" (internal structures)**  |
| **Glass Slide**         | 1.52                       | Background medium                     |
