# Kernset-monoflorale honing

Dit is de eerste kernsetlaag: honing die je in de praktijk koppelt aan een dominante drachtplant.

Je vindt hier de herkenningsprofielen voor monoflorale honingtypes.
Sommige profielen staan technisch onder *verdiepingset-herkenning*, maar horen inhoudelijk bij deze stap.
