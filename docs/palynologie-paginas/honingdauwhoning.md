# Honingdauwhoning

## Honingdauw elementen
- schimmel hyphen
- schimmel sporen
- unicellulaire algen
- pollen van nectarloze planten

- honingdauw kan gemaakt worden door diverse zuigende insecten op een aantal verschillende bomen

- Metacalfa pruinosa (citrus flatid planthopper) een mot geeft honingdauwhoning met specifiek kenmerken
    - eind 1970 geintroduceerd in Europa
